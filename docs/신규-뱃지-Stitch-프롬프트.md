## 📁 파일 저장 가이드

Stitch에서 생성된 PNG는 두 폴더에 동일 이름으로 넣기:

```
surf-wave-mobile/assets/images/badges/{KEY}.png
surf-wave-frontend/public/badges/{KEY}.png
```

저장 후 압축 스크립트 실행:

```bash
node D:/workspace/scripts/optimize-badges/optimize.js
```

> 모바일은 `BADGE_IMAGES` 객체에 `require()` 추가 필요:
> `surf-wave-mobile/src/screens/mypage/MyPageScreen.tsx` (현재 65~125번 줄)

---

## ✅ 작업 흐름

1. **이 문서를 보면서 Stitch에 16개 프롬프트 입력**
2. **각 PNG 다운로드** → 파일명을 위 KEY로 정확히 매칭 (예: `DIARY_50.png`)
3. **두 폴더에 복사**
4. **압축 스크립트 실행**
5. **백엔드** `badge-definitions.ts`에 16개 정의 추가 (별도 작업)
6. **백엔드** `badges.service.ts`에 트리거 로직 작성 (별도 작업)
7. **모바일** `BADGE_IMAGES`에 16개 require 추가 (별도 작업)
8. **운영 DB seed 동기화**

이미지 받으면 알려줘. 그때 6~8번 백엔드 + 모바일 코드 작업 같이 진행할게.
